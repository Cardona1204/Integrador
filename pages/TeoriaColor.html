Teoría del Color

Paleta de Colores utilizada para el diseño de la interfaz de usuario.

#1A1A1A - Se utiliza para elementos interactivos como botones, texto, bordes, como fondo de la metáfora de homicidios, como metáfora del hurto y parte del logo.
#E0E0E0 - Se utiliza para el community background.
#FFFFFF — Se utiliza para el fondo de cada registro de denuncia, barra de búsqueda y texto secundario de botones.
#D32F2F — Se utiliza para metáfora del homicidio, hace parte del mapa de calor, iconos de videos, para marcar la denuncia como falso y hace parte del logo.
#FFA000 — Se utiliza para metáfora del hurto, hace parte del mapa de calor y como fondo de la metáfora de hurto.
#2E8B57 — Se utiliza hace parte del mapa de calor y para marcar la denuncia como verdadera.
